# 《写个故事》决策速查

## 三书共识优先级

1. 人物选择与作品真正的价值判断。
2. 欲望、因果、冲突与高潮。
3. 类型承诺和观众体验。
4. 幕、段落、场景的推进功能。
5. BS2 节拍与页码检查。
6. Logline、受众与市场表达。

后两项不能破坏前四项。完整比较见 [three-book-synthesis.md](three-book-synthesis.md)。

## 先判断你卡在哪一层

| 症状 | 优先检查 | 立即动作 |
|---|---|---|
| 只有氛围或观念 | 主题 | 写“谁必须做什么，否则失去什么” |
| 人物很酷但不动 | 欲望 | 给出明确欲望对象与失败代价 |
| 开场全是说明 | 激励事件 | 先打破平衡，再在冲突中释放背景 |
| 场景像聊天 | 场景目标 | 写双方想从对方得到什么；标出价值前后态 |
| 中段重复 | 鸿沟/段落 | 改变结果类型，提高代价或改写目标 |
| 反派扁平 | 反面力量 | 加入制度、关系、内心及负面之负面 |
| 反转像作弊 | 伏笔/因果 | 提前建立事实，让新解释事后必然 |
| 高潮只有大场面 | 危机选择 | 写两难选项，让结果来自人物选择 |
| 主题像说教 | 主控思想 | 用高潮价值＋原因表达，删除宣讲台词 |
| 剧本不可拍 | 文本 | 把判断改写成行为、物件、空间或声音 |
| 概念说不清 | Logline | 写主人公＋行动＋阻力＋代价＋反讽 |
| 前提没有兑现 | Fun and Games | 补足观众为此概念而来的关键场景 |
| 中点后仍匀速 | Turn, Turn, Turn | 提高时限、公开性、代价或内外夹击 |
| 解说压住场景 | Pope in the Pool | 让信息在动作或关系冲突中释放 |
| 配角混在一起 | Limp and Eyepatch | 区分欲望、声音、策略与可见特征 |

## 短篇（10秒—10分钟）

1. 第一秒/第一页建立异常对照或即将失衡的日常。
2. 只保留一个主人公、一个主要欲望、一个核心价值。
3. 用一次激励、一次策略升级、一次决定性转折构成骨架。
4. 让重复动作发生意义变化，而非增加设定。
5. 结尾完成一个不可逆价值变化；谜底可以开放，作品不能未完成。

## 长篇

1. 主题句：人物＋主要行动。
2. 先定高潮选择，再反推开端。
3. 定激励事件、情节点 I、幕中重大逆转、情节点 II。
4. 每幕拆成有阶段目标和结果的段落。
5. 每场写目标、阻力、策略、转折、价值变化。
6. 初稿先完成；改写按“结构→人物→场景→对白”分层进行。

## BS2 十五节拍（商业长片检查用）

Opening Image → Theme Stated → Set-Up → Catalyst → Debate → Break into Two → B Story → Fun and Games → Midpoint → Bad Guys Close In → All Is Lost → Dark Night of the Soul → Break into Three → Finale → Final Image。

先问每个节拍是否真的改变行动或价值，再问页码。短片只保留所需功能，不压缩复刻十五格。

## 三种结构怎么选

| 需要 | 结构倾向 | 警告 |
|---|---|---|
| 强因果、强追看、明确结局 | 大情节/经典设计 | 不要机械套页码 |
| 心理、日常、暧昧余韵 | 小情节/最小主义 | 被动外部行动须由内在变化补偿 |
| 现实规则本身受质疑 | 反情节/反结构 | 混乱必须承载意义 |

## 场景五问

1. 谁想得到什么？
2. 谁或什么阻止他？
3. 人物换了几次策略？
4. 哪个结果击穿了预期？
5. 结尾哪个价值与开场不同？

## 最终质量闸门

- 高潮是否只能由这个人物在经历这段故事后作出？
- 反面力量是否足以逼出人物真相？
- 是否有场景只解释、不改变任何东西？
- 惊奇是否有因果与伏笔支撑？
- 主题是否存在于行动和后果，而非台词口号？
- 删除最漂亮的设定后，故事的人物选择是否仍成立？
