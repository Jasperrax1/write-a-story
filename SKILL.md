# 写个故事

**来源**：罗伯特·麦基《故事》；悉德·菲尔德《电影剧本写作基础》；Blake Snyder《Save the Cat!》  
**结构**：44章 | **更新日期**：2026-06-25

## 使用方式

- 用户只有想法：保护原始冲动，先形成“人物＋行动＋代价”，再选择结构。
- 用户要写短篇：使用短篇路径，不机械压缩120页三幕范式。
- 用户要写长篇：从高潮反推开端，建立幕、段落与场景卡。
- 用户提供剧本：先诊断问题层级，再给定向改写；不要一次重写所有层级。
- 用户问理论：读对应章节；需要快速判断时先读 [cheatsheet.md](cheatsheet.md)。
- 用户要比较三位作者：先读 [three-book-synthesis.md](three-book-synthesis.md)，把三书共识视为高置信默认，把 Snyder 页码视为商业长片诊断工具。
- 未给足创作信息时，优先追问人物欲望、失败代价、结尾价值；不要用随机情节点替用户决定作品核心。

## 三书共识（最高优先级）

三位作者反复重叠的内容优先于任何单一术语或页码：

1. 故事必须能压缩成一个人物采取行动并承担代价。
2. 主人公需要主动欲望；阻力必须迫使策略、风险和理解升级。
3. 结构的核心是行动转向和价值变化，不是固定页码。
4. 每场都要有目标、冲突、策略变化和可辨的前后差异。
5. 人物通过压力下的选择成立，弧光由开端与结尾的行动差异证明。
6. 高潮必须回答核心问题，并由主人公的最高代价选择完成。
7. 剧本以可见、可听的行动表达，不用解释替代戏剧。
8. 先用大纲或卡片验证结构，再写精致文本；结构工具应允许被发现推翻。
9. 类型是对观众的体验承诺，创新发生在兑现方式而非拒绝传统。

若规则冲突，依次保护：人物与价值 → 欲望/因果/冲突/高潮 → 类型承诺 → 幕段落场景 → BS2 页码 → 市场表达。完整依据与差异见 [three-book-synthesis.md](three-book-synthesis.md)。

## 核心工作流

### 1. 把冲动变成可写前提

保留用户最初被什么吸引。然后写：

> 一个【具体人物】必须【采取行动】，否则将失去【具体价值】；但【反面力量】迫使他面对【更深矛盾】。

“孤独、记忆、爱、梦”是观念，不是行动线。不要过早用参考作品或范式覆盖用户的原始冲动。

### 2. 确定主人公与故事价值

- 主人公要有可追踪的有意识欲望，也可以有与之冲突的无意识需要。
- 人物真相由高压选择揭示，不由履历、形容词或自我说明揭示。
- 选一个核心价值轴：真相/谎言、亲密/隔绝、自由/控制、安全/危险等。
- 暂写主控思想：“【最终价值】得以发生，因为【高潮选择的原因】。”它是可修订假设，不是先验口号。

### 3. 选择结构契约

- **大情节**：主动主人公、清晰因果、外部冲突、闭合结局。适合强追看和明确任务。
- **小情节**：压缩外部行动，强调内在冲突、多主人公或开放结局。每场仍需价值变化。
- **反情节**：主动破坏因果、时间或现实一致性。只在破坏本身承载意义时使用。
- 三幕范式是长片规划模型：建置—对抗—结局，由情节点连接。页码是参考，不是规则。
- BS2 是约110页商业长片的十五节拍检查表；先验证每个节拍的功能，再检查位置。短片不复刻十五格。

### 4. 建立行动骨架

1. **开端状态**：主人公目前如何生活，缺了什么。
2. **激励事件**：彻底打破平衡，唤起主要欲望。
3. **渐进复杂化**：每次行动产生不同于预期但可信的结果，迫使策略升级。
4. **情节点/幕高潮**：改写目标、风险或理解，把行动推入下一阶段。
5. **危机**：最高风险下不可兼得的选择。
6. **高潮**：选择造成不可逆价值变化，并回答核心戏剧问题。
7. **结局**：展示余波，收束必要材料，不另起更大的高潮。

### 5. 构建段落与场景

长篇先按阶段目标拆成段落。每个场景必须能回答：

1. 谁此刻想得到什么？
2. 谁或什么阻止他？
3. 人物用了哪些不同策略？
4. 哪个结果击穿预期？
5. 结尾哪个价值与开场不同？

若没有价值变化，场景通常只是解说或过场。把必要解说放进隐瞒、交换、逼问、误解和选择中。

### 6. 写成可拍文本

- 用现在时、主动语态写可见行动。
- 把心理判断转成行为、物件、空间、光线或声音。
- 对白是人物争取结果的策略；优先写潜文本，删除对已演出的感受说明。
- 镜头指令只在它决定观众何时获得关键信息时使用。

### 7. 分层改写

依次处理：核心行动与高潮 → 人物选择与反面力量 → 幕和段落 → 场景转折 → 对白、描写与格式。先解决上层问题；结构改变后再润色句子。

## 短篇路径

适用于10秒至10分钟：一个主人公、一个主要欲望、一个核心价值。第一秒/第一页建立视觉或行动上的异常对照；用激励事件、一次升级和一次决定性转折形成骨架。结尾必须完成价值变化，但可以保留谜底。短不是信息少，而是变量少、转折准。

## 长篇路径

先写高潮选择与最后动作，再设计与之对照的开端。确定激励事件、两个主要情节点和幕高潮；每幕拆成段落，每段落以明确结果触发下一段。先用卡片验证行动链，再写处理稿和正式剧本。

## 关键判断规则

- 当人物只有设定时，用压力选择揭示真相。
- 当中段发软时，不要只加障碍；改变目标、冲突层级、代价或现实理解。
- 当反转像作弊时，补足伏笔与因果，使其“意外但必然”。
- 当反派扁平时，把反面力量扩展到关系、制度、环境和人物内心。
- 当主题说教时，删除结论台词，让正反价值都在事件中获得证据。
- 巧合可制造早期困境，不应在高潮替人物解决困境。
- 类型惯例必须兑现，表面实现必须更新；反类型不是不完成。
- 开放结局可以不回答全部事实，但必须完成核心体验和价值判断。

## 范围与限制

本 skill 提炼三本书的理论，不替代对类型作品、真实世界和目标制片格式的专项研究。菲尔德与 Snyder 的页码范式主要针对传统商业长片；处理短片、剧集、互动叙事或实验作品时，保留结构功能，按媒介重新调整尺度。Snyder 的市场判断来自特定工业语境，应作为传播与节奏工具，不应覆盖作者真正的价值选择。
